# 2025-Yishu1-1
2025 FYP Lightweight Agile Dashboard
1. 在自己的電腦上創建一個 .env 文件（請在 backend 資料夾中進行）
詳細：
1.1 backend terminal： 
-------------------------------------------------
npm install dotenv
-------------------------------------------------
1.2 .env（注意前面有個點，而且沒有副檔名：
-------------------------------------------------
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=你那個很私人的密碼
DB_NAME=agile_dashboard
PORT=3000
-------------------------------------------------
2. 在 backend 資料夾下，檢查有沒有一個叫 .gitignore 的文件。如果沒有，就建立一個。
   然後打開 .gitignore，在裡面新增一行：
-------------------------------------------------
.env
node_modules/
-------------------------------------------------
